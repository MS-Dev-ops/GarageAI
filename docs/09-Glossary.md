\# Glossary



AI Capability



A business feature powered by AI.



Conversation



Natural language interaction between a user and the system.



AI Orchestrator



The central gateway responsible for coordinating AI workflows.



Provider



Technology implementation that communicates with an AI model.



Examples



OpenAI



Azure OpenAI



Ollama



Embedding



A numerical representation of text used for similarity search.



RAG



Retrieval-Augmented Generation.



Combines AI with external knowledge.



Tool Calling



Allows an AI model to execute business operations.



Prompt



Instructions supplied to an AI model.



Token



A unit of text processed by an AI model.



Context Window



The maximum amount of information an AI model can consider during a request.



Vertical Slice



An architectural style where each feature owns its commands, queries, handlers and models.



DDD



Domain Driven Design.



CQRS



Command Query Responsibility Segregation.
